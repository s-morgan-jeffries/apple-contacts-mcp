# Security Policy

## Supported Versions

This project is in pre-release alpha. Only the latest published version receives security fixes.

| Version | Supported |
|---------|-----------|
| 0.x     | Latest only |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via [GitHub private security advisory](https://github.com/s-morgan-jeffries/apple-contacts-mcp/security/advisories/new).

**Do not open a public issue for security vulnerabilities.**

### Response Timeline

- **Acknowledgment:** Within 48 hours
- **Fix or mitigation:** Within 7 days

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Security Model

See [docs/guides/SECURITY_CHECKLIST.md](docs/guides/SECURITY_CHECKLIST.md) for the security checklist applied to every feature. A full threat model will be added before v0.1.0.
